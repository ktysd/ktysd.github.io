    Python/Colab 超入門

    ※ Colab の利用には，Google (Gmail) のアカウントが必要です．

    数値計算で使用するプログラミング言語 Python と，その実行環境 Colab の使い方を速習します．

    • Python とは，人気沸騰中のプログラミング言語で，汎用性の高さがウリです．これまで複数のプログラミング言語を組み合わせないと書けなかった処理が，これ1つで書けてしまいます．AI分野の標準言語としても有名です．
    • Colab (Google Colaboratory) とは，Google が無償提供している，クラウド型の Python 実行環境です．
    • Python 自体は，単体で PC にインストールして使うこともできますが，ここではメディア授業を考慮して，Colab 経由でブラウザから実行する方法を学びます．
